# 📊 Sales Analysis Dashboard | Power BI

## 📌 Project Overview

This project analyzes sales performance using Power BI to help stakeholders make data-driven business decisions through an interactive dashboard.

---

## 🎯 Project Objectives

- Analyze overall sales and profit performance.
- Identify top-performing products and customers.
- Evaluate sales across different regions.
- Analyze the relationship between discounts and profitability.
- Identify loss-making products.
- Build an interactive dashboard for business users.

---

## 🛠️ Tools Used

- Power BI
- Power Query
- DAX
- Data Modeling

---

## 📂 Data Preparation

- Removed duplicate records.
- Handled null values.
- Changed data types.
- Loaded cleaned data into Power BI.
- Created relationships between tables.

---

## 📈 KPIs Created

- Total Sales
- Total Profit
- Profit Margin
- Total Orders
- Total Customers

---

## 📊 Dashboard Features

- KPI Cards
- Bar Chart
- Line Chart
- Pie Chart
- Interactive Slicers

---

## 💡 Business Insights

- Technology products generated the highest sales revenue.
- High discounts negatively affected profitability.
- A small group of customers contributed the majority of revenue.
- Some products generated losses despite high sales.
- Sales trends indicated future growth opportunities.

---

## ✅ Business Recommendations

- Review discount strategy.
- Focus marketing on profitable products.
- Improve customer retention for high-value customers.
- Monitor loss-making products regularly.
